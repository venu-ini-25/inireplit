Build a production-style web application called iNi (Invent N Invest).

PRODUCT OVERVIEW
iNi is a unified AI-powered strategic finance platform for early-stage to growth-stage companies and investors. The product connects data from multiple business systems, standardizes and consolidates the data, and provides real-time dashboards, financial reporting, board and investor reporting, KPI tracking, portfolio management, and M&A workflows.

The platform should serve two main user groups:
1. Businesses / operators
2. Investors / portfolio managers

Primary stakeholders:
- CFO
- CEO
- Finance team
- Board members
- Investors / operating partners
- Corp dev / M&A users
- External advisors with limited access

CORE PRODUCT GOALS
- Connect fragmented systems into one finance-ready platform
- Provide real-time insights and faster reporting
- Reduce manual consolidation and repetitive finance work
- Support investor reporting and board reporting
- Support portfolio monitoring across multiple companies
- Provide M&A due diligence and deal workflow support
- Enable secure collaboration with role-based access and NDA gating
- Track verified data status and last-audited timestamps
- Blend software + advisory workflows so human finance support can coexist with the SaaS platform

TARGET CUSTOMER PROFILE
- Companies from roughly $1M to $100M revenue for core operating use
- Investors and businesses in the lower middle market / growth stage
- C-level executives, finance teams, board partners, investors
- Portfolio businesses and deal teams

TECH STACK
Use this stack unless there is a strong reason to change:
- Frontend: Next.js + React + TypeScript
- UI: Tailwind CSS + shadcn/ui
- Charts: Recharts
- Tables: TanStack Table
- State: React Query + Zustand
- Backend: Node.js with Next.js API routes or Express
- Auth: Clerk or NextAuth
- Database: PostgreSQL via Prisma
- File storage: Supabase Storage or AWS S3
- Background jobs: Inngest or BullMQ
- Data sync/integrations layer: modular connector architecture
- AI layer: OpenAI APIs for summaries, anomaly explanations, commentary, and report drafting
- Hosting: Replit deployment compatible
- Design: clean executive SaaS product, premium, modern, finance-first

DESIGN STYLE
- Premium B2B SaaS
- Clean, minimal, dark/light theme support
- Dashboard-driven
- Executive-friendly
- Strong information hierarchy
- Large KPI cards, high-quality tables, drill-down charts
- Feel like a blend of Rippling + Ramp + Carta + modern BI, but built for finance and M&A workflows
